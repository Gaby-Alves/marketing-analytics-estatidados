# marketing-analytics-estatidados
Este repositório tem como objetivo armazenar o projeto final do curso de extensão em Marketing Analytics oferecido pelo Estatidados e a UFG, que foi ministrado pelos grandes mestres prof. Thiago Marques e prof. Marcos Severo.

A base de dados a ser analisada é do Airbnb com os dados do Rio de Janeiro, e este repositório contará com scripts com o código, um dashboard em Power BI e o report que é o objeto que será avaliado pelos professores a fim de obter o certificado de conclusão deste curso de Extensão;
